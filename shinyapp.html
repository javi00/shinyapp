Shiny App

You can visit my shiny app on this links Wildfires. This app includes a map with wildfires in Bolivia.

Also you can download the project in my github

I used two libraries:

library(shiny)
library(leaflet)

The ui.R file

I used the follow funtions in this part

            sliderInput(" ")
            checkboxInput(" ")
            div(" ",style = "color:blue")

mainPanel:
           
            leafletOutput(" ")  # to view the map
            h3(" "),
            h4(" "),
            textOutput(" ")

the server.R file

I used the follow funtions in server part

   reactive({ })
   renderLeaflet({ })  # to plot a map 
   
   leaflet()
   addTiles() 
   setView()        # to  view a specific country 
   addCircles()     # to add the points
   
   renderText({ })
